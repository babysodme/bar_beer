# 呼家楼之驴

风味描述：酸甜

器皿:

  • 苟市啤酒大杯

原材料:

  • 伏特加 60ml
  • 柠檬汁 30ml
  • 干姜水
  • 柠檬皮

制作步骤:

  1. 放入4块方冰，搅拌勺旋转冰杯、滤水
  2. 依次加入伏特加、柠檬汁、搅拌10秒充分混合
  3. 倾斜杯子 缓慢倒入干姜水
  4. 吧勺触杯底旋转1-2圈、提冰块两次
  5. 柠檬皮擦杯装饰

# 玛格丽特

风味描述：甜口、草莓水果香

器皿:

  • 搅拌机
  • 金属杯

原材料:

  • 草莓2颗
  • 龙舌兰agavales 60ml
  • 草莓糖浆 10ml
  • 盐
  • 碎冰1杯

制作步骤:

  1. 杯边放在柠檬肉上旋转沾上柠檬汁、杯口浸入盐里、沾边、敲杯震去多余盐边
  2. 将草莓、碎冰、龙舌兰、草莓糖浆加入搅拌杯，搅拌至没有咔咔声
  3. 倒入杯中、四分之一块草莓装饰

# 海明威知道

风味描述：酸甜系\好喝\酸甜口;海明威喜欢喝 daiqur 戴奇丽——海明威自杀那天喝的;中东酒：咸酸奶+茴香酒

器皿:

  • 小小玻璃杯

原材料:

  • 朗姆酒Havana club 40ml
  • 柠檬汁 20ml
  • 接骨木糖浆sarklass 15ml
  • 茴香酒Pernod （一吧勺）

制作步骤：

  1. shaker壶加入朗姆、柠檬汁、糖浆、茴香酒、加4-5块方冰
  2. 摇壶至表面结霜
  3. 5毫米的柠檬皮条装饰（可用吧勺盘卷）

Tips：
壶盖盖拔两次保持内外气压一样再摇

# 金汤力

风味介绍：清爽系

器皿:

  • 细圆柱玻璃杯

原材料:

  • 金酒 45ml
  • 汤力水

制作步骤：

  1. 杯中放入4块方冰、吧勺搅拌冰杯、滤冰器滤水
  2. 依次倒入金酒、汤力水倒满
  3. 吧勺延边触底旋转1-2圈、向上提两次冰块、压一下冰块
  4. 柠檬皮装饰

# 老烟枪

重口系

器皿:

  • 威士忌杯

原材料:

  • 拉夫格威士忌 30ml
  • 咖啡酒 20ml

制作步骤:

  1. 杯中置入两块方冰、搅拌冰杯、滤冰器倒水
  2. 一次倒入拉夫格、咖啡酒、再次搅拌
  3. 肉桂一头点火、燃烧一头向上放置杯中

Tips:
燃烧一侧的肉桂可以在杯内绕两圈因为这样子看起来很帅

# 蹦个养生迪

风味介绍：甜口系

器皿: 马提尼杯

原材料:

  • 红酒 30ml
  • 红枣酒半斗
  • 橙汁 30ml
  • 半颗红枣
  • 橙皮

制作步骤:

  1. 杯中置入两块方冰、搅拌冰杯、滤冰器倒水
  2. 依次倒入红酒、橙汁、红枣酒轻轻搅拌
  3. 放入半颗红枣、橙皮装饰

Tips

  1. 倒酒： 酒标朝前、手托底在下部三分之一处
  2. 搅拌动作：中指无名指夹住吧勺、用中指往回拉 无名指往前推
  3. 杯子拿下边不要破坏杯子温度
